# pose_similarity_test

Assuming you have a dataset of pairs of  two skeletons (as 2 lists of pairs of numbers) and you  have a score of "similarity" between each two

The task is to train a model \ find an algorithm to predict "similarity" of a new pair of two skeletons

Задача решалась следующим образом:
1) был собран игрушечный датасет с точками скелета 1 человека.
2) рандомно выбиралось два положения человека, в которых подсчитывались углы между определенными точками тела(это сделано для того,
чтобы уйти от зависимости координат от различных длин конечностей/позиции съёмки человека) и составляеться новый датасет как разница между углами.
3) оценка похожести подсчитывалась с помощью косинусного расстояния между скелетами, хотя и не совсем корректна так как не содержит в себе важность о пложении определённых конечностей.
4) для того чтобы сеть обучалась дынные были нормализованы с помощью StandardScaler
5) для обучения выбрана линийная модель с mse лосом
6) сеть переобучаеться
